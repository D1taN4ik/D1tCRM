# D1tCRM

Современная CRM-система для управления сотрудниками, проектами и задачами.

## Возможности

- Управление сотрудниками
- Система ролей и авторизации
- Управление проектами
- Постановка и контроль задач
- Профили сотрудников
- PostgreSQL + Entity Framework Core
- Docker и Docker Compose
- Адаптивный интерфейс
- Стекломорфизм и современный UI

---

# Технологии

- ASP.NET Core 10
- Blazor Server
- Entity Framework Core
- PostgreSQL
- ASP.NET Identity
- Docker
- Docker Compose

---

# Роли в системе

## Администратор

- Полный доступ к системе
- Управление сотрудниками
- Управление проектами
- Управление задачами

## Менеджер

- Управление проектами
- Управление задачами
- Просмотр сотрудников

## Сотрудник

- Просмотр своих проектов
- Просмотр своих задач
- Доступ к профилю

---

# Запуск проекта

## Требования

- Docker Desktop

---

## Запуск

```bash
docker compose up --build
````

После запуска приложение будет доступно по адресу:

```text
http://localhost:8080
```
